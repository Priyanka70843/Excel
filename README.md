# Excel Project
# 📊 Business Data Analysis & Reporting

## Overview
This project focuses on cleaning, transforming, and analyzing business opportunity data to create meaningful reports and insights. The dataset contains project information, opportunity close dates, currencies, amounts, and other business metrics.

The goal of this project is to prepare raw business data for reporting by adding calculated fields, performing currency conversions, and organizing data for further analysis.

---

## Dataset

The dataset includes the following information:

- Record ID
- Project Name
- Opportunity Close Date
- Currency
- Amount
- Total Line Items
- Total Closed Live Product Items
- Total Closed Dropped Items
- Project Duration
- Opportunity Created Date

---

## Tasks Performed

- ✔️ Cleaned and organized the dataset
- ✔️ Added helper columns
  - Opportunity Created Month
  - Opportunity Created Quarter
  - Close Won Week
  - Close Won Month
  - Close Won Quarter
- ✔️ Converted transaction amounts into USD
- ✔️ Prepared data for business reporting
- ✔️ Structured data for dashboard creation and future analysis
